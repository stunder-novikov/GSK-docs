# Quick Start

## 1. Enable Plugin

Edit -> Plugins -> Game Settings Kit -> Enable. Restart the editor.

## 2. Enable Enhanced Input User Settings

Project Settings -> Enhanced Input -> **Enable User Settings** = true.

Required for key remapping. Without this setting, remapping will be unavailable.

## 3. Open Example

Open the map `Content/GameSettingsKit/Example/Maps/L_SettingsExample`. Press Play, press Escape -- you'll see a fully working settings menu with all categories: Display, Graphics, Audio, Input, Gameplay.

Explore the example: Data Assets in `Example/Data/`, widgets in `Example/Widgets/`, Blueprints in `Example/Blueprints/`. Use as reference for integrating into your own project.
